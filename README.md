# FocusFlow – Smart Pomodoro Timer

## Project Description
FocusFlow is a smart Pomodoro Timer built using HTML, CSS, and JavaScript. It helps users manage tasks, prioritize work, and improve productivity using structured focus sessions.

## Tech Stack
- HTML
- CSS
- JavaScript
- Local Storage

## How to Run
1. Download the project
2. Open `index.html` in any browser

## Target Users
Students, developers, writers, professionals.
